### 2.2 各竞品优劣势

#### React 19
- **优势**：生态最成熟、社区最大、用工市场最广
- **劣势**：Hooks 规则复杂、虚拟 DOM 性能开销、34% AI 生成错误率

#### Vue 3.6
- **优势**：学习曲线平缓、单文件组件体验好、Vapor Mode 性能提升
- **劣势**：.value 上下文切换、Options/Composition API 撕裂

#### Svelte 5
- **优势**：编译时优化、语法简洁、无运行时
- **劣势**：Runes 引入造成分裂、响应式依赖静态分析局限

#### Solid 1.9
- **优势**：细粒度更新、性能优秀
- **劣势**：读写分离语法噪音、生态不成熟

### 2.3 Aether 差异化定位

**核心差异化**：AI-First Design
- 不是为了替代现有框架，而是为 AI 代码生成场景优化
- 编译时极简 + 运行时极小 = AI 容易理解、容易生成
- 内置所有必要功能，消除选择疲劳
