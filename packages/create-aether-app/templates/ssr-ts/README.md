# My Aether SSR App

A server-side rendering project with TypeScript created with [Aether](https://github.com/zelixag/aether) - AI-Friendly Reactive Framework.

## Getting Started

```bash
npm install
npm run dev
```

## SSR Development

This template includes server-side rendering support with full TypeScript type checking.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run typecheck` - Run TypeScript type checking
- `npm run build:server` - Build SSR bundle
