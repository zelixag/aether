# My Aether App

A TypeScript project created with [Aether](https://github.com/zelixag/aether) - AI-Friendly Reactive Framework.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run typecheck` - Run TypeScript type checking
